# Trace

A lightweight client communication and project update tool built for agencies and freelancers.
Project leads create a project, post updates, and share a unique link with their client. Clients can view updates and leave comments in real time, no account required. When the project is complete, the lead closes it and all associated data is cleared.
Built with: Next.js, TypeScript, Supabase

## Features

- Project creation with unique shareable client link
- Real time update posting and client commenting
- Email notifications on project creation and new updates
- Automatic data cleanup on project close
- No client account required, access via unique link only
- Built with multi-tenant licensing in mind
